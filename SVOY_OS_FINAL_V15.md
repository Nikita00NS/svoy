# СВОЙ OS v15 — Полный продукт управления ТГ-каналами (работает 24/7 пока ты отдыхаешь)

## Идея (вишенка на торте 🍒)

**СВОЙ — это не просто админка, а Авто-ОС для сети каналов:**

- Бот @SVOIROBOT — сам принимает заявки на рекламу, новости, апелляции, работу, поддержку (нижние кнопки Reply Keyboard)
- Авто-модерация — удаляет мат, спам, флуд, банит — пока ты спишь
- Авто-писатель — берет RSS ленты, переписывает через бесплатную нейронку Groq/Gemini (без карты), ставит ватермарку, сам публикует в нужный канал
- Админка — красивая, без технических надписей, показывает статусы, комменты модератора, статистику канала, выручку с рекламы
- Работает везде и всегда: Railway бэкенд + воркеры + Redis + Postgres — 24/7

## Что теперь работает (чего не было раньше)

### 1. Публикация постов из админки (было сломано)
- Вкладка Контент → Создать пост/Опрос: выбираешь канал, пишешь текст, выбираешь тип TEXT/PHOTO/VIDEO/POLL
- Для опроса: вопрос + варианты (кнопка + вариант)
- Кнопки: Создать черновик / Создать и опубликовать сразу (прямой вызов Telegram API, не через очередь)
- Если канал не выбран — даст понятную ошибку, не 500
- Публикация поддерживает фото, видео, опросы через `sendPoll`

### 2. Потребитель опубликовал пост → статус и комменты модератора
- Пользователь в боте жмет "Предложить новость" → в админке появляется во вкладке Заявки тип NEWS_PROPOSAL
- Админ меняет статус: NEW → IN_REVIEW → APPROVED/REJECTED + пишет Комментарий модератора (поле теперь есть)
- Комментарий и статус автоматически улетают клиенту в бот: "✅ Одобрена / ❌ Отклонена + Комментарий модератора: ..."
- Кнопка "→ В контент" превращает заявку в пост

### 3. Создание опросов и многое другое
- В контенте можно выбрать mediaType POLL, ввести вопрос и варианты
- Публикуется через `sendPoll` с анонимностью

### 4. Подключение источников к определенному каналу
- Вкладка Источники (RSS) → при создании выбираешь "В какой канал постить?" + чекбокс "Авто-публиковать сразу"
- При Fetch: новости переписываются через AI (Groq/Gemini/OpenRouter бесплатно) и если autoPublish=true и канал выбран — сразу ставятся в SCHEDULED и публикуются воркером
- Стало: 1 RSS = 1 канал, всё привязано

### 5. Статистика и настройки канала
- Вкладка Каналы → кнопка 📊 Статистика → дергает Telegram API `getChat` + `getChatMemberCount` + считает кол-во постов
- Сохраняет в `statsJson`
- Вкладка канала: название, handle, ID, описание, ватермарка, переключатели Авто-модерация / Авто-публикация RSS, settingsJson для badWords

### 6. Бот оснащен механиками (авто)
- Авто-модерация включена если в канале `autoModeration=true`
- Проверяет: мат (из settingsJson.badWords), >3 ссылок = спам, >5 сообщений за 10 сек = флуд → MUTE/BAN/DELETE
- Логирует всё в Модерация
- Принимает заявки 5 типов и создает Intake + Content
- Полноценный Reply Keyboard внизу (не инлайн)

### 7. Продажа рекламы (вишенка для заработка)
- Заявки типа AD_ORDER — это заказы на рекламу
- В дашборде считается выручка: кол-во APPROVED * $150 (заглушка, можно поменять)
- В Интейках видишь бюджет, желаемую дату, можешь одобрить и сразу создать пост-рекламу
- Всё работает пока ты отдыхаешь: клиент пишет боту "Купить рекламу" → ты одобряешь в админке → пост уходит в канал → деньги

## Нейросеть бесплатно и без ошибок (вопрос пользователя)

**НЕ бери OPENAI_API_KEY** — требует карту, дает 429.

**Бери бесплатные, без карты, которые я уже подключил в AiService (auto-выбор):**

1. **Groq — рекомендую** (самый быстрый, 500 ток/сек)
   - Сайт: https://console.groq.com
   - Вход через Google, без карты
   - Create API Key → `gsk_...`
   - В Railway: `GROQ_API_KEY=gsk_...` + `AI_PROVIDER=groq`
   - Лимиты: 1000 req/day, 30 RPM, бесплатно вечно
   - Модель: `llama-3.1-8b-instant`

2. **Google Gemini — самый щедрый**
   - https://aistudio.google.com → Get API Key
   - Без карты, 1500 req/day
   - В Railway: `GEMINI_API_KEY=...` + `AI_PROVIDER=gemini`
   - Модель: `gemini-2.0-flash`, 1M токенов контекст

3. **OpenRouter — много бесплатных моделей**
   - https://openrouter.ai → Keys → Create (без карты)
   - `OPENROUTER_API_KEY=sk-or-...` + `AI_PROVIDER=openrouter`
   - Модель: `deepseek/deepseek-r1:free` (бесплатная рассуждающая)

В коде `AiService.detectProvider()` сам выберет: сначала Groq, потом Gemini, потом OpenRouter, потом OpenAI. Можешь поставить все ключи сразу.

После добавления ключа — кнопка AI Rewrite в контенте и авто-переписывание RSS заработают без ошибок.

## Что дальше (как работать и продавать рекламу)

1. Создай каналы в админке (у тебя уже СВОЙ Луганск есть в БД, сделай `SELECT *` показал 1 ряд — теперь он появится после v15)
2. Добавь @SVOIROBOT админом во все каналы
3. Добавь RSS: lenta.ru/rss + выбери канал + включи авто-публикацию
4. Поставь GROQ_API_KEY в Railway → AI начнет переписывать новости бесплатно
5. Бот уже принимает рекламу: клиент жмет "Купить рекламу" → ты видишь в Заявках бюджет → одобряешь → создаешь пост → публикуешь → получаешь деньги. Всё пока ты отдыхаешь.
